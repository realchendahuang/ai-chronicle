2019 年，OpenAI Five 在公开表演赛中面对 OG——2018 年 Dota 2 国际邀请赛冠军。赛制是三局两胜；OpenAI Five 连下两局取胜。五个智能体各控制一名英雄，在同一支队伍里实时协作。这一句结果必须和限制条件一起读：比赛使用约 17 名可选英雄池，并对物品与部分游戏机制施加约束，并非当时完整天梯的无限制英雄池与规则全集。

Dota 2 与围棋的差别写在环境本身。时间连续，动作空间巨大，视野部分可观测，一局可达数十分钟，胜负依赖五人资源分配、拉野、抢符、团战与买活时机。OpenAI 用大规模分布式强化学习与自我对弈堆经验：公开材料称系统每天积累约相当于 180 年的对局时间量级（按人类游玩时间折算的宣传口径，应理解为数量级描述）。算法侧是近端策略优化等策略梯度方法的大规模变体，配以重放与工程化的训练基础设施；奖励塑造与观测特征高度针对 Dota 规则。

公开胜局证明：在经过裁剪但仍极复杂的实时团队游戏里，纯自我对弈可以练出击败世界冠军战队的团队策略——至少在该版本规则与英雄池内如此。它没有证明强化学习已经“解决”电子竞技，更没有证明样本成本可被任意任务承受。数百万年量级的模拟对局换来的是一个环境专用智能体；换一张地图、改一套补丁，往往意味着重新燃烧计算。部分可观测与长程信用分配仍然困难，人类选手的适应速度与迁移能力并不在同一成本曲线上。

训练基础设施包括大量 GPU 与 CPU 组成的滚动手，观察空间由游戏 API 提供的结构化特征构成，而不是原始像素。英雄池限制降低了组合爆炸，使自我对弈能够在有限动作集合里反复磨练阵容理解。公开赛使用的补丁版本与英雄列表被记录在技术博客中；脱离这些约束去宣称“AI 统治 Dota”，会超出证据范围。真正可推广的观察是：复杂团队游戏可以被切开，在切片上用算力换策略。

OpenAI Five 的位置介于 AlphaGo 的棋盘清晰性与开放世界的混乱之间。它把强化学习的叙事从回合制完美信息，推到必须沟通站位与技能时机的团战；也把“算力换数据”的逻辑推到公共视线：没有便宜的模拟器，这条路走不远。表演赛结束后，系统与代码的完整复现门槛仍然极高。留下的是一盘盘在规则围栏内打出来的胜局，以及围栏本身——17 名英雄与其它限制提醒读者，复杂环境的突破常常先发生在被工程切开的切片上。

与棋类项目相比，Five 的工程比例更高：特征、奖励、课程与算力调度缺一不可，算法名称反而没有架构图那么醒目。公开胜利证明切片后的复杂环境可以被攻克，也证明切片本身是结果的一部分。后来的游戏人工智能与多智能体研究继续引用其自我对弈与大规模训练经验，同时更谨慎地讨论样本效率与迁移。一百八十年每天的折算数字适合传播，适合提醒成本，不适合单独支撑“通用智能已至”的结论。

部分可观测意味着每个智能体必须在战争迷雾下维持对未知区域的信念，团队奖励则带来信用分配难题：一次成功的团战，功劳如何回溯到几分钟前的买活与眼位。工程上通过奖励塑造与统计特征缓解，但并未从理论上取消难题。公开赛胜利展示的是在给定约束下的策略强度，展示不了的是换规则后的迁移成本。读 Five，应同时读英雄池列表与每天折算对局年。

实时游戏把强化学习的样本饥渴放大到公共数字：每天约一百八十年。数字适合做标题，也适合做警告。警告的内容是：没有模拟器与规则切片，这条路的账单会更难看。Five 的胜局写在约束之内，约束本身必须写入史实。

十七名英雄与物品限制不是脚注，而是结果成立的条件。去掉条件再夸大胜局，是宣传，不是历史。把条件写进正文，胜局才仍有技术含量。
