15.3%，26.2%。

ILSVRC 2012 的 top-5 测试错误率把两个数字摆在同一行：AlexNet 的集成模型是前者，第二名是后者。十点九个百分点的差距大到很难再解释为调参运气。计算机视觉研究者已经见过卷积神经网络——LeNet 一类网络在更早的数字与小规模识别任务上就存在——也知道 GPU 擅长并行计算；Fei-Fei Li 团队推动的 ImageNet 也已提供超过百万张带标签训练图像。条件散落在论文、硬件手册与数据集网站上，谁也没有被成绩表单独说服。2012 年秋天这张表突然说明：当数据、并行计算与足够深的卷积被当成同一个系统来训，得到的不是小幅改良，而是别人用旧配方难以追上的分差。

Krizhevsky、Sutskever 与 Hinton 在 NeurIPS 2012 论文《ImageNet Classification with Deep Convolutional Neural Networks》里给出的网络有五个卷积层和三个全连接层，约六千万个参数、六十五万个神经元。训练图像约 120 万张，另有 5 万验证、15 万测试；图像先缩放到 256×256，再截取 224×224 的输入。分类目标是 1000 个 ImageNet 类别。与竞赛中常见流水线不同，系统不再先手工设计 SIFT、HOG 之类局部描述，再接词袋、编码与支持向量机，而是让卷积核直接从像素统计里形成层级特征。作者并没有写“特征工程已死”；他们把设计重心从“提取什么边缘或纹理”挪到结构、损失、正则、数据增强与硬件约束上。特征仍然存在，只是改由梯度下降在数据里长出来。

没有单一技巧能独占功劳。ReLU 使深层网络比 tanh 等饱和激活训练得更快——论文用 CIFAR-10 对照显示，达到相同训练误差所需的迭代次数显著更少。数据增强包括随机裁剪、水平翻转，以及在 RGB 通道上做 PCA 扰动，以扩大有效样本、减轻过拟合。全连接层使用概率 0.5 的 Dropout：训练时随机关闭一半神经元，测试时恢复完整网络。测试阶段对四角与中心共五处裁剪及其水平翻转、合计十个视图取平均预测。局部响应归一化等细节也被写入；后来未必全部成为标准，但在 2012 年它们与整体配方绑定在一起，复现时不能只抄架构图而丢掉训练日程。

硬件限制写进了架构图本身。单块 GTX 580 只有 3GB 显存，完整模型放不进去，于是网络按通道切开，分到两块 GPU 上；多数时候两卡各算一半通道，只在特定层交换信息。约九十个 epoch，在两块 GTX 580 上训练五到六天。竞赛提交进一步使用七个模型集成，才得到 15.3% 的 top-5 错误率；单模型已经明显低于第二名，集成把差距拉得更刺眼。论文末尾写得很直白：网络主要受 GPU 内存和研究者愿意等待多久的限制；更快的 GPU 与更大的数据还会继续抬高上限。那不是空泛的规模崇拜，而是从装不下模型的机器里逼出来的判断。

竞赛之后数年，视觉研究与工业系统迅速把“手工特征加浅分类器”换成可端到端训练的卷积栈。ImageNet 提供标注规模，CUDA 与消费级 GPU 提供可负担的矩阵吞吐，ReLU、Dropout 与增强提供让大模型可训的细节。公司实验室开始按“能否在 ImageNet 上接近或超过 AlexNet”来筛选方向；博士论文的默认基线也从编码加 SVM 转向卷积网络。后来的 VGG 用更深的 3×3 堆叠换精度，GoogLeNet 用 Inception 模块换效率，ResNet 用捷径解决退化——它们都要先站在这张成绩表已经改写的坐标系里。

竞赛组织与评估协议也值得一提。ILSVRC 使用 top-5 错误率：模型给出的五个类别猜测中只要包含正确标签即算对，这一指标比 top-1 更能反映大规模细粒度分类的难度，也让 15.3% 与 26.2% 的对比集中在同一套规则下。作者公开了架构与训练细节，使其他实验室可以在相近硬件上尝试复现；“可复做”本身成为结果说服力的一部分。

AlexNet 赢下的是一次分类竞赛，交出来的却是一组可复做的条件清单：足够多的标注图像、装得下模型的并行硬件、端到端卷积、以及一整套让六千万参数不至于完全过拟合的细节。争论可以继续围绕某一层的宽度、某一种归一化或某一项增强。表格本身不参与争论。它只是把数据、并行计算与深层卷积第一次以无法用修辞缩小的分差，钉在了同一行里：15.3 对 26.2，中间是 10.9 个百分点。

两块消费级显卡、五到六天、七模型集成——这些数字今天读来近乎寒酸，却构成了当时能够完成的实验边界。ImageNet 的类别长尾、细粒度差异与拍摄条件变化，使手工特征很难用同一套描述符覆盖全部一千类；卷积网络用同一套可学习滤波器扫描整图，把覆盖问题交给数据与容量。top-5 协议下的十五点三，并不是“机器已经看懂世界”的证明，而是旧基线在同一把尺子上被远远抛开的证明。此后竞赛与论文摘要开始默认报告卷积结果，手工特征流水线逐渐从主舞台撤到对比实验的第一行。
