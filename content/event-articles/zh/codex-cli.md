2025 年 4 月 16 日，OpenAI 在 o3 与 o4-mini 发布节奏里抛出 Codex CLI：开源、本机终端、读仓库、改文件、跑命令。名字继承 2021 的 Codex，产品却是 2025 的 Agent 语法——与 Claude Code 同一信任域：开发者的 shell。

云端 Codex 任务队列解决并行与隔离；CLI 解决同步与本机反馈。两者并列，才补齐 OpenAI 的双轨。开源客户端让权限、沙箱与配置可被审查，也让官方终端 Agent 不再是 Anthropic 一家的故事。

与 Cursor、Devin 对照：编辑器收上下文，沙箱收任务，终端收失败日志。Codex CLI 的史位是把第三格贴上 OpenAI 商标。选型从此更像选信任域，而不是选唯一赢家。SWE-bench 仍在旁冷眼看所有宣传。
