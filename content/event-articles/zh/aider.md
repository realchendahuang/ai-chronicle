在 Claude Code 与 Codex CLI 成为商标之前，Aider 已经在终端里做一件很土的事：聊天、出 diff、写进 Git。模型可以换，仓库纪律不能换。Apache 许可让自带密钥的人把结对带进真实项目。

它的史位是范式而非峰值能力。没有融资发布会，只有文档与提交记录。后来所有“本机 Agent”宣传，都可以在 Aider 的工作流里找到祖先：映射代码库、约束补丁、留下可回滚历史。

Agent Coding 专题若只从 Devin 讲起，会漏掉这截开源前史。终端三角的第三条边，早有人用 Git 画过。
