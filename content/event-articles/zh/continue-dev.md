Continue 把聊天与补全塞进 VS Code / JetBrains 侧栏，并坚持模型可换、流程可配。当 Cursor 用融合 IDE 讲述未来时，Continue 讲述另一种：你的编辑器可以留下，模型可以自选。

开源使企业能审计与自托管。能力起伏随后端模型而变；产品哲学钉在“不锁仓”。2026 年前后被 Cursor 收购，反而证明开放侧栏曾足够重要——值得买下，而不是忽视。

Agent Coding 地图上，Continue 是编辑器开源层。没有它，商业 IDE 的对照会少一根坐标轴。
