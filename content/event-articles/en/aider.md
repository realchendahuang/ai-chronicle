Before Claude Code and Codex CLI became trademarks, Aider already did something plain in the terminal: chat, emit diffs, write into git. Models can swap; repo discipline cannot. Apache terms let BYOK users bring pairing into real projects.

Its seat is paradigm, not peak capability. No fundraising keynote—docs and commits. Later “local agent” pitches find ancestors in Aider’s flow: map the codebase, constrain patches, leave roll-back history.

If an agent-coding topic starts only at Devin, it misses this open prehistory. Someone already drew the third side of the terminal triangle with git.
