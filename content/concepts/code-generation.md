id: code-generation
title: 代码生成（Code Generation）
fullName: Code Generation
difficulty: intermediate
oneLiner: 由模型根据自然语言、注释或上下文自动写出可运行或可修改的程序代码。
beginnerExplanation: 代码生成可以是补全一行函数、根据描述写出脚本，也可以是跨文件的重构与脚手架。Codex、GitHub Copilot 等把“在海量代码上预训练的语言模型”接到了编辑器与 API 中。生成结果仍需测试、审查与权限控制，不能默认正确。
whyItMatters: 代码生成把编程中重复与样板工作部分自动化，改变了人写软件的协作方式与效率基线。
relatedEvents:
  - codex
  - github-copilot
relatedConcepts:
  - llm
  - coding-agent
  - ai-pair-programmer
  - token
commonMisunderstandings:
  - 生成能编译或通过测试，不代表设计合理、安全或可维护。
  - 代码生成不等于完整的软件工程 Agent；后者还包含规划、工具与验证闭环。
